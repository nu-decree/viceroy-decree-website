VICEROY DECREE scholars can take courses from a shared virtual institute curriculum. These courses follow a virtual online instructional method for Visiting VI students. Each scholar needs to complete 8–10 credits from this multi-institution curriculum (course credits vary from 3 to 4 credit hours).

{{< image src="images/viceroy-decree-course-path.png" caption="" alt="alter-text" height="" width="" position="center" command="fill" option="q100" class="img-fluid" title="image title" webp="false" >}}

Depending on scholars' background they can select courses that complement their skills and knowledge based on breadth or depth.

- Courses that complement the student's existing background and skills
- Courses that amplify and extend the student's existing background and skills

Course curricula is divided into the following areas:

- Electromagnetic Spectrum
- Cybersecurity
- Cryptography
- Data Science
- Strategic Foreign Language
- Research

---

## DoD Cyber Workforce Framework (DCWF) Alignment

All VI courses are mapped to the **DoD Cyber Workforce Framework (DCWF)** as defined in **DoDI 8140.02**, which builds directly on the NICE Workforce Framework for Cybersecurity. The tables below list the NICE Framework Knowledge IDs and corresponding DCWF Work Roles covered by each course. This mapping is maintained by the Virtual Institute and updated as NCAE-C designation requirements evolve.

**DCWF Work Role Categories covered across the VI curriculum:**

| Category | DCWF Symbol | Work Roles Covered by VI Courses |
| -------- | :---------: | -------------------------------- |
| Securely Provision | SP | Software Developer, Secure Software Assessor, Systems Requirements Planner, Information Systems Security Developer, Systems Developer, Research & Development Specialist |
| Operate and Maintain | OM | Database Administrator, Data Analyst, Network Operations Specialist, System Administrator, Systems Security Analyst |
| Oversee and Govern | OG | Security Control Assessor, Information Systems Security Manager, Authorizing Official, Program Manager, IT Project Manager, Executive Cyber Leadership |
| Protect and Defend | PD | Cyber Defense Analyst, Cyber Defense Infrastructure Support Specialist, Cyber Defense Incident Responder, Vulnerability Assessment Analyst |
| Analyze | AN | Multi-Disciplined Language Analyst, All-Source Analyst |
| Collect and Operate | CO | Target Developer, Target Network Analyst, Cyber Ops Planner |
| Investigate | IN | Cyber Defense Forensics Analyst, Law Enforcement / Counterintelligence Forensics Analyst |

*Offered in the Fall and Spring*

Prerequisite(s): CS 2510 or DS 2500

**DS 3500 Advanced Programming with Data (4 credits)**

*Offered in the Fall and Spring*

Description: Provides a deep-dive into the design and implementation of enterprise-grade software systems with an emphasis on software architectures for complex data-driven applications. Covers extensible architectures supporting testing, data provenance, reuse, maintainability, scalability, and robustness; building software APIs and libraries. Students learn to design, implement, and test loosely-coupled service-oriented architectures using Python for distributed processing, stream-based data processing, and inter-process communication via message-passing.

Prerequisites: DS 2000, DS 2500

Syllabus can be found [here](/files/NUDS3500.pdf).

Prerequisite: CS 205, CYB 136, MAT 226, and (STA 270 or STA 275) with grades of C or better in each. Letter grade only.
