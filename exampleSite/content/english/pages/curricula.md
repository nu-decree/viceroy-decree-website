VICEROY DECREE scholars can take courses from a shared virtual institute curriculum. These courses follow a virtual online instructional method for Visiting VI students. Each scholar needs to complete 8–10 credits from this multi-institution curriculum (course credits vary from 3 to 4 credit hours).

{{< image src="images/viceroy-decree-course-path.png" caption="" alt="alter-text" height="" width="" position="center" command="fill" option="q100" class="img-fluid" title="image title" webp="false" >}}

Depending on scholars' background they can select courses that complement their skills and knowledge based on breadth or depth.

- Courses that complement the student's existing background and skills
- Courses that amplify and extend the student's existing background and skills

Course curricula is divided into the following areas:

- Electromagnetic Spectrum
- Cybersecurity
- Cryptography
- Data Science
- Strategic Foreign Language
- Research

## DoD Cyber Workforce Framework (DCWF) Alignment

All VI courses are mapped to the **[DoD Cyber Workforce Framework (DCWF)](https://public.cyber.mil/wid/dod8140/)** as defined in **DoDI 8140.02** and **DoDM 8140.03**, which build directly on the NICE Workforce Framework for Cybersecurity. Each course table below lists the NICE Framework Knowledge IDs, corresponding DCWF Work Roles and codes, DCWF category, and approved CompTIA certifications per **[DoDM 8140.03](https://www.comptia.org/en-us/resources/comptia-framework-alignment/)**.

Work role links reference the **[NICCS NICE Framework](https://niccs.cisa.gov/workforce-development/nice-framework)**. DCWF codes and approved certifications reference the **[DoD 8140 Qualification Matrices](https://public.cyber.mil/wid/dod8140/qualifications-matrices/)** and the **[CompTIA Framework Alignment page](https://www.comptia.org/en-us/resources/comptia-framework-alignment/)**.

**DCWF Work Role Categories covered across the VI curriculum:**

| Category | Symbol | Description | Reference |
| -------- | :----: | ----------- | :-------: |
| Securely Provision | [SP](https://public.cyber.mil/wid/dod8140/) | Design, build, procure, or deliver secure IT systems | [DoD 8140](https://public.cyber.mil/wid/dod8140/) |
| Operate and Maintain | [OM](https://public.cyber.mil/wid/dod8140/) | Provide support, administration, and maintenance of IT systems | [DoD 8140](https://public.cyber.mil/wid/dod8140/) |
